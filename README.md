## ğŸ§© System Architecture
1. IoT sensors collect temperature and location data.
2. Sensor data is received and processed by a **Node.js + Express backend**.
3. Validated data is recorded as transactions on the **Hyperledger Fabric blockchain**.
4. A frontend dashboard displays real-time sensor readings.
